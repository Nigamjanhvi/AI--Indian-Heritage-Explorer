# IMAGE FOLDER SETUP GUIDE

## 📸 Adding Images to Your Website

This website is designed to be image-rich. All images should be placed in the `images/` folder.

### Required Images

Below is the complete list of image filenames referenced in the website. You can download free, high-quality images from:
- **Unsplash**: https://unsplash.com/
- **Pexels**: https://pexels.com/
- **Pixabay**: https://pixabay.com/

### 🌟 HERO CAROUSEL IMAGES (PRIORITY - 7 Images)

These are the main full-screen images that appear on the home page hero carousel:

1. **taj-mahal.jpg** - Taj Mahal (Full view, preferably during golden hour)
2. **red-fort.jpg** - Red Fort Delhi (Main structure view)
3. **kashi.jpg** - Kashi Vishwanath Temple (Temple view or Ganga ghat)
4. **golden-temple-hero.jpg** - Golden Temple Amritsar (Preferably with reflection)
5. **meenakshi-hero.jpg** - Meenakshi Temple Madurai (Colorful gopuram towers)
6. **hampi-hero.jpg** - Hampi ruins (Ancient stone chariot or temple complex)
7. **india-gate-hero.jpg** - India Gate Delhi (Monument view, preferably evening)

**Image Specs for Hero Carousel:**
- **Resolution**: 1920x1080px minimum (Full HD landscape)
- **Aspect Ratio**: 16:9 (widescreen)
- **Format**: JPG
- **File Size**: 300-800KB (compress for web)
- **Quality**: High quality, sharp, well-lit photos
- **Orientation**: Horizontal/Landscape ONLY

**Tips for Hero Images:**
- Choose dramatic, inspiring photos
- Good lighting (golden hour preferred)
- Clear subject with some background
- Avoid cluttered or busy backgrounds
- Higher resolution is better (will be scaled down)

### Image Specifications

**Recommended Sizes:**
- Hero images: 1920x1080px (landscape)
- Carousel images: 1200x600px (landscape)
- Card images: 800x600px
- Gallery images: 800x600px
- General content images: 600x400px

**Format:**
- JPG or JPEG (preferred for photos)
- PNG (for images requiring transparency)
- Max file size: 500KB per image (compress if needed)

### Quick Setup Steps

1. Create the `images` folder in your project root
2. Download images from free stock photo sites
3. Rename them to match the filenames above
4. Place them in the `images` folder
5. Refresh your website to see the images
